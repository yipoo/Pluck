最新

v0.1.0

2026-06-21

首个公开版本 — 完整覆盖截图 OCR / 剪贴板历史 / 标注 / 长截图(实验)。

核心功能

  1. 全局热键 Control + Option + A 区域截图,Apple Vision 自动 OCR(中文 95%+ 准确率)
  2. 全局热键 Control + Option + V 打开剪贴板历史(LIKE 子串搜索,中英文)
  3. 菜单栏常驻 (LSUIElement),无 Dock 图标
  4. 多屏环境智能识别鼠标所在屏幕,Retina + 外接显示器混合处理正确
  5. 截图视觉反馈:被截区域 200ms 白光淡出 + 系统截图音(AudioToolbox 沙盒友好)
  6. 动态菜单栏图标三态(就绪 / 启动中 / 截图中,带 SF Symbol pulse 动画)
  7. 4 Tab 设置面板:通用 / 热键 / 隐私 / 关于
  8. 首次启动 3 步引导:隐私承诺 / 热键 / 权限

历史窗口

  1. NavigationSplitView 3 栏布局(侧栏筛选 + 卡片列表 + 详情)
  2. 侧栏类型筛选:全部 / 文本 / 图片 / 文件 / 截图历史,带计数
  3. 截图缩略图(ImageIO 高效解码 + NSCache 缓存)
  4. 截图详情:大图 + 完整 OCR 文字双栏并排 + 复制 / 导出 / Finder / 删除

标注画布(v0.2 提前实现)

  1. 4 种工具:矩形 / 箭头 / 高亮 / 文本
  2. 8 色调色板 + 3 档线宽
  3. 撤销 / 重做 / 一键导出 PNG
  4. 归一化坐标存储,缩放预览不失真

长截图(v0.2 实验性)

  1. ImageStitcher 算法:垂直拼接,SSD 灰度采样找重叠最优 offset
  2. 浮窗式手动滚动 UI(空格抓下一屏 / 回车完成)
  3. 真实场景测试中(滚动 App 边界处可能有错位)

工程基础

  1. 零外部依赖(仅 Apple 原生框架:SwiftUI / AppKit / Vision / ScreenCaptureKit / SQLite3)
  2. macOS 14+,Apple Silicon + Intel 通用二进制
  3. App Sandbox + Hardened Runtime + Developer ID 签名 + 公证
  4. 13 个单元测试(Models / Storage / OCR / Settings)
  5. "检查更新"入口已就绪(Sparkle SPM 接好后自动激活)

隐私承诺

  1. 默认不申请网络权限,Activity Monitor 可验证零外联
  2. 不内嵌任何分析、崩溃、广告 SDK
  3. 所有 OCR / 历史 / 截图本地处理
  4. PrivacyInfo.xcprivacy 仅声明 UserDefaults(理由 CA92.1)
  5. 严格离线模式开关(隐私 Tab,默认开启)
