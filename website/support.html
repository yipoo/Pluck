🚀 快速开始

第一次用 Pluck?

  1. 下载并打开 Pluck — 它会出现在菜单栏(顶部状态栏)
  2. 系统会请求"屏幕录制"权限,授予后即可使用
  3. 按 ⌃⌥A 拖动选择屏幕任意区域 → 文字会自动识别并写入剪贴板
  4. 按 ⌃⌥V 打开剪贴板历史窗口

❓ 常见问题

截图无反应?

99% 是屏幕录制权限。检查:

  1. 系统设置 → 隐私与安全性 → 屏幕录制
  2. 找到 "Pluck" 并勾上
  3. 完全退出 Pluck(⌘Q,不是关窗口)
  4. 重新启动

如还不行,在终端跑:

tccutil reset ScreenCapture com.yipoo.Pluck

然后重启 Pluck,系统会重新弹权限对话框。

OCR 识别不准?

  • 印刷体中文 / 英文准确率 95%+
  • 手写体目前不太行(v0.3 计划支持)
  • 低分辨率或截区太小可能影响 — 试着选大一点

历史搜索找不到?

  • 历史默认保留 100 条;可在 设置 → 通用 调到 1000
  • 搜索是子串匹配,中英文都支持
  • 类型过滤:左侧栏点 文本 / 图片 / 文件 缩小范围

菜单栏图标不见了?

系统屏幕太小或图标被挤出去了。两种解法:

  • 装 Bartender / Hidden Bar 工具,把不常用的菜单栏图标隐藏起来
  • 退出 Pluck 重新打开,会重新出现在菜单栏

能跨设备同步吗?

v0.1 不支持。我们隐私优先,数据只存本机。v2.0 会加 iCloud 同步(可选,默认关)。

能用在 Windows / iPad 吗?

  • iOS 配套 App 计划在 v1.0(2027 Q1)
  • Windows 版计划在 v2.0(2027 Q3)
  • 当前仅 macOS 14+

  • 当前没有任何收费计划

🔐 数据完全删除

如果你想完全卸载并清除所有数据:

  1. 在 Pluck 设置 → 隐私 → 清空所有剪贴板历史 + 清空所有截图
  2. 退出 Pluck
  3. 把 Pluck.app 拖到废纸篓
  4. (彻底)删除 ~/Library/Containers/com.yipoo.Pluck/

由于我们不存任何数据在云端,这些步骤就是全部。
