隐私政策

最后更新:2026-04-27

1. 我们的承诺(TL;DR)

Pluck 不收集你的任何数据。

  • 所有 OCR / 截图 / 剪贴板历史 只存在你的 Mac 上
  • App 默认 不申请网络访问权限(可在 Activity Monitor 验证)
  • 不内嵌任何分析、崩溃、广告 SDK
  • 不收集设备 ID、IP、邮箱、姓名 等任何识别信息
  • 没有用户账号体系,无需注册登录

2. 我们处理什么数据

2.1 截图(Screenshots)

当你按 ⌃⌥A 选择屏幕区域,Pluck 调用 Apple ScreenCaptureKit 截屏。截图文件保存在:

~/Library/Containers/com.yipoo.Pluck/Data/Library/Application Support/Pluck/snapshots/

这些文件 从不离开你的设备。Pluck 不发送任何截图到服务器。

2.2 OCR 文字识别

Pluck 用 Apple Vision Framework 在本机识别截图中的文字。Vision 是 macOS 系统级 API,识别过程完全本地,Apple 也不会上传图像。

2.3 剪贴板历史(Clipboard History)

Pluck 监听系统剪贴板变化(通过 NSPasteboard 轮询)并存到本机 SQLite 数据库:

~/Library/Containers/com.yipoo.Pluck/Data/Library/Application Support/Pluck/pluck.sqlite

默认保留最近 100 条(可在设置调整)。这些内容只在你的 Mac 上。

对于标记为敏感的内容(密码管理器写入的 org.nspasteboard.ConcealedType),Pluck 主动跳过不存储。

2.4 用户偏好(UserDefaults)

设置面板的偏好(热键、外观、历史保留条数等)存储在 macOS 标准 UserDefaults。这些是设备本地数据,不会上传到任何服务器。

3. 我们 不 做的

  • 不发送数据到云端
  • 不与第三方分享数据
  • 不内嵌追踪 / 分析 / 广告 SDK
  • 不使用 Cookie(我们不是网站,但官网也只用必要 Cookie)
  • 不要求账号 / 注册 / 登录
  • 不读取你 Mac 上其他文件
  • 不连接任何远程服务器(除你显式开启 Sparkle 自动更新检查)

4. 苹果 Apple 官方 API 用途说明

Pluck 使用以下需要"使用原因"声明的 Apple API:

  • NSPrivacyAccessedAPICategoryUserDefaults:CA92.1 — 仅访问 App 自身的偏好设置(热键 / 外观)

其他 API(Vision / ScreenCaptureKit / NSPasteboard)是常规使用,无需特殊声明。

5. 屏幕录制权限

macOS 14+ 强制要求 App 申请"屏幕录制"权限以截屏。Pluck 在首次截图时会触发系统对话框,你可以在 系统设置 → 隐私与安全性 → 屏幕录制 随时撤销。

这个权限 只用于截图功能本身,不会用于其他用途。

6. 软件更新检查

如果你在 设置 → 通用 中点 "检查更新",Pluck 会向 https://pluck.yipoo.com/appcast.xml 发送一次 HTTPS 请求,获取最新版本号。仅此一次,且仅你主动触发。我们的服务器只看到一次匿名 HTTPS 请求,不存储任何 IP / User Agent 用作分析。

"自动检查更新"默认 关闭。

7. 数据删除

你可以随时:

  • 设置 → 隐私 → 清空所有剪贴板历史(清数据库 + 磁盘 PNG)
  • 设置 → 隐私 → 清空所有截图
  • 直接卸载 Pluck → 删除 ~/Library/Containers/com.yipoo.Pluck/

由于我们不存任何数据在云端,卸载即彻底清除。

8. 儿童隐私

Pluck 不针对 13 岁以下儿童设计或推广,但任何年龄使用都安全 — 我们不收集数据。

9. 第三方服务

当前版本 不集成任何第三方服务。未来可选 AI 增强功能(v0.3+)接入备案 LLM(豆包 / Kimi / DeepSeek)时,会:

  • 默认 关闭,需用户显式开启
  • 开启时弹出明确告知("此功能会把选区文字发送到 X 服务")
  • 每次使用前再次确认
  • 各 LLM 服务商的隐私政策由其自行公布

10. 中国法律合规

Pluck 严格遵循中国《个人信息保护法》(PIPL)及《数据安全法》。由于我们不收集任何"个人信息",大多数条款不适用。

如未来需要扩展数据处理范围,我们会:

  • 更新本隐私政策并通知用户
  • 申请相应资质(如需要)
  • 保持"不上传敏感信息"的核心承诺

11. 政策变更

本政策的任何重大变更会:

  • 提前 30 天在官网首页公告
  • 在 App 内通知现有用户
  • 用户可选择不接受新版,继续使用旧版本(不强制升级)
