常见问题

需要登录账号吗?

不需要。Pluck 完全本地运行,没有账号体系。下载即用。

能上 App Store 吗?

v0.1 走自有官网下载(已 Developer ID 签名 + 公证)。Mac App Store 在评估,但沙盒限制可能影响截图体验。

支持 Windows 吗?

v1 仅 macOS 14+。Windows 版在 v2 路线图(预计 2027 Q3)。

真的完全免费?会不会以后收费?

是的,完全免费,无功能限制、无水印、无广告、无内购解锁。当前不打算收费。后续可能加一个"打赏"按钮(类似 Buy Me a Coffee),纯自愿,不影响任何功能可用性。

OCR 中文准确率怎么样?

基于 Apple Vision Framework,印刷体中文 95%+,与 macOS 系统截图 OCR 同水平。手写体在 v0.3 计划中。

能不能把数据导出?

剪贴板历史存在本地 SQLite,可直接用 sqlite3 命令行打开。截图 PNG 在 ~/Library/Application Support/Pluck/snapshots/。完全可移植。
