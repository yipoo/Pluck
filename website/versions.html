v0.2.0 预计 2026-09 发布,含马赛克工具 / 自动滚动长截图 / 自定义热键录制 UI。详见 规划。

系统兼容性

  1. macOS 14 (Sonoma):已测试,完整功能
  2. macOS 15 (Sequoia):已测试,完整功能
  3. macOS 26 (Tahoe / 后续):开发与测试主力
  4. macOS 13 (Ventura) 及更老:不支持(MenuBarExtra 与 ScreenCaptureKit 14+ 特性必需)

架构

所有版本都是通用二进制(Universal Binary),Apple Silicon (M1 / M2 / M3 / M4 / M5) 与 Intel Mac 都能跑。Apple Silicon 上 OCR 性能约为 Intel 的 3 倍(Vision Framework 走 Neural Engine)。

校验下载完整性

下载后可用 shasum 校验文件未被篡改:

shasum -a 256 ~/Downloads/Pluck-0.1.0.dmg

对照 GitHub Release 页面公布的 SHA-256 哈希(发布后会同步)。

自动更新

已安装的 Pluck 会通过 Sparkle 框架检查更新(默认关闭自动检查,可在 设置 → 通用 → 检查更新 主动触发)。订阅更新源:appcast.xml

归档老版本

Pluck 会保留所有公开版本,旧版本永久可下载。如果新版有兼容性问题,你可以随时回退。

(目前没有归档版本 — 0.1.0 是首发版。)
